# Falling blocks game
You are a square and have to dodge falling blocks.

To move the square, use on of the following controls:

- Left and Right arrow keys
- a (move left) and d (move right) keys
- Click and drag mouse, the square will follow.

Some falling blocks are collectable and give you temporary powers.

The special blocks have different colors. Here is a list with all blocks explained:

- Cyan: When collected, it gives you a temporary shield; You are immune to falling blocks.
- Red: When collected, it temporarily makes you slower; It's harder to dodge the falling blocks.
- Light green: When collected, it temporarily makes you faster; It's easier to dodge the falling blocks.
- Orange: When collected, all falling blocks on the screen will disapear.
- Yellow: When NOT collected, ten additional blocks will spawn and fall down, so you better collect it.
- Pink: This block can be any of the above. It's random.

There are two game modes.

In the game mode "auto level-up", the level increases automatically as you play longer without getting hit by a block.

In the game mode "manual level-up", you have to collect the gray-blue blocks to increase the level by 1.

The higher the level, the more blocks fall down, so it gets harder.

Have fun!
